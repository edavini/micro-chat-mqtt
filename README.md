# MQTT Based Micro Chat
A Micro chat built with MQTT over NodeJS.  
DB used is Mongo with mongoose

## What's inside

* NodeJS Express APP

* MongoDB
* MongoExpress
* Mongoose
* Angular
* Angular Material
* [**NGX-MQTT**: A simple MQTT.js wrapper](https://github.com/sclausen/ngx-mqtt)
